感谢下载巽式连续音录音表（罗马音辅助版）。
此录音表可用于OREMO和wavSpitz。

OREMO配布地址：http://freett.com/nwp8861/soft/
原版下载地址：https://ch.nicovideo.jp/tatsu3/blomaga/ar426004


■与原版的区别
·将原录音表用于提示下个发音的comment替换为辅助阅读的罗马音
·追加各录音表的纯罗马音版本（不保证可用于oto自动推算）
·追加特殊发音录音表的comment

使用这个录音表，可以在录音时参考罗马音的同时仍保持录制好的文件名为假名。

■内容物
reclist文件夹		：录音表文件夹
└〇〇mora.txt		：假名版录音表
└〇〇mora(romaji).txt	：罗马音版录音表
└〇〇mora-comment.txt	：辅助阅读的罗马音comment

reclist(vowel first)	：录音表文件夹（优先元音排列的版本）
└同上

reclist(extra)		：特殊音素录音表文件夹
└extra.txt		：录音表（仅假名版）
└extra-comment.txt	：comment文件
└extra oto templet.ini	：特殊音素oto.ini

hira2roma list.txt	：假名→罗马音对应表
readme(original).txt	：日文原版readme
readme.txt		：本文件


■使用方法
打开OREMO之后选择[ファイル]→[音名リストの読み込み]，
选择对应的录音表文件即可。

在选择假名版录音表后一般会自动读取comment，
若未正常显示请尝试选择[コメントファイルの読み込み]。

若仍不能正常显示，请确认OREMO的版本高于oremo-3.0-b190106。

■How to use
Open OREMO, choose[File]→[Load voice list],
Then choose the reclist file.

OREMO will load the comment file too.
If it did't, please try the [Load comment file].

Please make sure OREMO's version is later than oremo-3.0-b190106.
